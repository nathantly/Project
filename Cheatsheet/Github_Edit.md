Some basics Github formatting syntax:
## Headings
Simply put # at the beginning of the sentence.
(#) is the biggest
(######) is the smallest

## Styling text
**Bold**        **  **   

*Italic*        *  *

~~Strikee through~~     ~~   ~~

<Subscript>       <   >

## Quoting text
Using >

Text that is not quoted
> Text that is quoted

## Quoting code
Using single backticks ` ` to `quote` within a sentence
or using ```   ``` to quote multiple sentences

```
git add .
git commit -m "Update"
git push
```
